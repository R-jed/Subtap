## 四、已知问题

### 4.1 时间轴对齐质量

**问题描述：** 对齐后的时间戳存在重叠，多条字幕起始时间相同（00:00:01,230）。

**影响：** 字幕显示效果不佳，需要优化对齐算法。

**建议：** 使用 `--mode quality` 或手动调整。

---

### 4.2 LLM 清洗依赖 Ollama

**问题描述：** 文本清洗阶段需要本地 Ollama 服务运行，否则降级为原文。

**影响：** 无法自动修复 ASR 错别字。

**建议：** 安装并启动 Ollama 服务，或配置其他 LLM 后端。

---

### 4.3 TUI 模式下进程卡死

**问题描述：** 在 TUI 模式下运行时，进程可能卡死（CPU 98.8%）。

**影响：** 用户体验不佳。

**建议：** 使用 `--no-tui` 参数避免此问题。
