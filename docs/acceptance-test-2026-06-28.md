## 五、已知问题

### 5.1 时间轴对齐质量

**问题描述：** 对齐后的时间戳存在重叠，多条字幕起始时间相同（00:00:01,230）。

**影响：** 字幕显示效果不佳，需要优化对齐算法。

**建议：** 使用 `--mode quality` 或手动调整。

---

### 5.2 LLM 清洗依赖 Ollama

**问题描述：** 文本清洗的 LLM 增强层需要本地 Ollama 服务运行。

**影响：** 无，LLM 为可选增强，本地清洗已作为基础保障。

**建议：** 安装并启动 Ollama 服务以启用 LLM 增强。

---

### 5.3 TUI 模式卡死

**问题描述：** 在 TUI 模式下运行时，进程可能卡死（CPU 98.8%）。

**影响：** 用户体验不佳。

**建议：** 使用 `--no-tui` 参数避免此问题。

---

### 5.4 质量报告模式描述过时

**问题描述：** 质量报告仍显示"快速模式（跳过清洗和对齐）"，但实际已不再跳过。

**影响：** 报告信息不准确。

**建议：** 更新 report.py 中的模式描述。
